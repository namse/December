csv 파일을 읽기 전에 꼭 utf-8형식으로 저장해야 한글이 깨지지 않는다.

Save json/csv 를 선택할 경우, json 파일이랑 csv 파일이 함께 저장된다.

Open csv (현재는 json 파일 읽는 모드는 없음...)
를 해서 편집하고자 하는 경우, 
원하는 cell을 선택하고 F2를 누르면 편집이 가능하다.

Add Item 버튼을 누르면 item 추가도 가능하며

아무 cell이나 우측 클릭을 누른 후 Add Column하면 항목도 추가가 가능하다.

아직 delete column은 구현하지 않았다.

일단 쓰기에 적당하고 생각될 정도만 구현했으므로
